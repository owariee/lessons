## Exemplo 2 - Parte 2

Dessa forma, poderíamos definir essas variáveis em C, sendo:
```cpp
int visao_por_personagem = 1;
int elementos = 7;
int elemento_por_visao = 1;
```
ou em Python, que é dinamicamente tipada:
```cpp
visao_por_personagem = 1
elementos = 7
elemento_por_visao = 1
```
